# ⚡ Photon Grid

> A modern, high-performance, framework-agnostic data grid built with TypeScript.

Photon Grid is an enterprise-grade data grid designed for handling everything from small datasets to millions of rows. It is built around a single TypeScript core and provides native wrappers for Angular, React, and Vue.

> 🚧 Photon Grid is currently under active development.

---

## ✨ Features

- 🚀 High-performance virtual scrolling
- 📊 Handles millions of rows
- 🔄 Sorting
- 🔍 Filtering
- 📑 Pagination
- 📌 Pinned columns
- 📐 Column resizing
- ↔️ Column reordering
- 🎯 Row & cell selection
- ✏️ Cell editing
- 🎨 Custom cell renderers
- 🧩 Framework agnostic core
- ⚡ Optimized rendering engine
- 📦 Tree-shakable packages
- 💻 TypeScript-first

# Installation

Install the package for your framework.

### Angular

```bash
npm install photon-grid-angular photon-grid-core
```

### React

```bash
npm install photon-grid-react photon-grid-core
```

### Vue

```bash
npm install photon-grid-vue photon-grid-core
```

# Development

Clone the repository

```bash
git clone https://github.com/abdulwahid-csit/photon-grid.git
```

Install dependencies

```bash
npm install
```

Run development

```bash
npm run dev
```

Build packages

```bash
npm run build
```
